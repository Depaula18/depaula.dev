## 💻 Sobre o Projeto

Este portfólio foi desenvolvido para refletir minha identidade como Desenvolvedor Full Stack (C# .NET + React) e atuar como um catálogo centralizado dos meus projetos, como o Game Catalog System. O foco arquitetural foi criar uma aplicação que não apenas funcione perfeitamente, mas que passe credibilidade técnica e maturidade de código.

### ✨ Principais Funcionalidades
- **Internacionalização (i18n):** Suporte nativo e fluido para Português (PT-BR) e Inglês (EN).
- **Integração Contínua (CI/CD):** Pipeline configurado com deploy automático via Vercel.
- **Banco de Dados Real-time:** Vitrine de projetos consumida diretamente do Firebase Firestore, permitindo atualizações de conteúdo sem necessidade de novos deploys.
- **Formulário de Contato Serverless:** Envio de e-mails diretos integrado com EmailJS.
- **Animações e Microinterações:** Uso extensivo de Framer Motion e conceitos de Glassmorphism.
- **Theming Dinâmico:** Sistema de troca de variáveis CSS (`blue`, `red`, `purple`) injetado via React State.

## 🛠️ Instalação e Execução Local

Caso queira clonar o repositório para testar a arquitetura na sua máquina:

1. Clone o repositório:
`git clone https://github.com/Depaula18/depaula.dev`

2. Entre na pasta do projeto:
`cd depaula.dev`

3. Instale as dependências:
`npm install`

4. Variáveis de Ambiente:
Crie um arquivo `.env` na raiz do projeto e preencha com as suas chaves do EmailJS e Firebase:
`VITE_EMAILJS_SERVICE_ID=your_service_id`
`VITE_EMAILJS_TEMPLATE_ID=your_template_id`
`VITE_EMAILJS_PUBLIC_KEY=your_public_key`

5. Rode o servidor de desenvolvimento:
`npm run dev`
